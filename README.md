# airline data repository
Simple flight data search website focused on api usage

## Tech Stack
- React
- TypeScript
- Vite
- HTML
- CSS
- JSON
- Material UI
## Features
- Serpapi api usage
- Material UI
## What I Learned
- Solutions for some bugs
- API handling
- More on Data Manipulation
## Running Locally
Go to https://serpapi.com, make a free account, and copy the api key into a .env file, call it VITE_API_KEY, then:
- npm install
- npm run dev
